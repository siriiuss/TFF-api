# Welcome to TFF Leagues Scraper

This is the official documentation for the **TFF Leagues Scraper**.

## Overview
Scrapes real-time football league standings from the official Turkish Football Federation (TFF) website.

### Key Features
* **Real-time Data:** Always up-to-date with TFF servers.
* **In-Memory Caching:** Extremely fast response times and protection against rate limits.
* **Resilience:** Built to handle TFF server timeouts and HTML structure changes gracefully.

Please navigate using the sidebar to explore endpoints and system architecture.
